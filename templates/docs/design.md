# Design template (compat stub)

> **Do not use this file as the design structure.**  
> Use `templates/docs/design/{s|m|l}.md` via `resolveDesignTemplate(pipeline)`.

Default for unknown custom pipelines with a design step: **`design/m.md`**.

Sub-document templates (`design-domain-model.md`, `design-database.md`, …) remain separate; see new-plan skill.
